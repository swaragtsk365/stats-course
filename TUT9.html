Model Selection and AIC

  • Ideally we would want to estimate out of sample error
  • Akaike called it an entropy maximization principle and minimizing AIC is equivalent to maximizing entropy in a thermodynamic system.
  • We will never know what the exact model is. Thus in the process of fitting a model, some information will be lost about the true process from which the data got generated.

Akaike Information Criteria(AIC)

  • AIC measures the relative information loss.
  • Since we dont know the exact model generating the data, we cant estimate the absolute information loss.
  • The idea is to compare the relative information loss among multiple models.
  • The likelihood of a model compared to the best model can be given as:
  • \(p(Model_{i}) = e^{\frac{(AIC_{best} - AIC_{i})}{2}}\)
  • \(AIC_{i} = 2k_{i} - 2log(L(\theta_{i}|y))\)

Intuition behind AIC

  • Let \(g(y)\) be the true function that is generating the data.
  • Let \(f(y|\theta_{k})\) be the candidate function with k parameters.
  • Let \(L(\theta_{k}|y)\) be the ikelihood of \(f(y|\theta_{k})\)
  • Objective: Find the \(f(y|\theta_{k})\) such that it approximate \(g(y)\) well. That is we should reduce out of sample error.

KullbackLeibler divergence

  • We would use the idea of KullbackLeibler divergence for proceeding.
  • A type of statistical distance: a measure of how one probability distribution P is different from a second, reference probability distribution Q.
  • For discrete probability distributions P and Q defined on the same sample space,{}, the relative entropy from Q to P is defined to be:
  • \(D_{KL} (P||Q) = E[log(\frac{P(x)}{Q(x)})] = \sum_{x \in X} P(x) log(\frac{P(x)}{Q(x)})\) ## AIC
  • Ideally we want lesser KL Divergence between \(f(y|\theta_{k})\) and \(g(y)\).
  • \(D_{KL} (g(y)||f(y|\theta_{k})) = E[log(\frac{g(y)}{f(y|\theta_{k})})]\) should be low.
  • \(E[log(\frac{g(y)}{f(y|\theta_{k})})] = E[log(g(y)) - log(f(y|\theta_{k}))]\)
  • \(D_{KL} (g(y)||f(y|\theta_{k})) = E[log(g(y))] - E[log(f(y|\theta_{k}))]\)
  • \(2D_{KL} = - 2E[log(f(y|\theta_{k}))] + 2E[log(g(y))]\)
